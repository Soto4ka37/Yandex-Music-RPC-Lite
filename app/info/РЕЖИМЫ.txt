# Режимы времени:
Внимение! Из-за ограничений API отображение времени трека не зависит от положения вашего плеера, если вы поставите трек на паузу или промотаете трек скрипт продолжит работать "проигнорировав" это.

maintime = 0
Отключает любое отображения времени трека. Только обложка, автор и название. 

maintime = 1
Используется только "Осталось [Длинна трека]". Если трек повторяется - таймер начинается с начала, а в статусе отображается что трек повторяется.

maintime = 2 (Рекомендовано)
Аналогично timemode = 1, но если трек повторяется "Остлось" меняется на "Прошло" и отсчитывает время, пока трек не изменится.

# Режимы волны:
wavetime = 1 (Рекомендовано)
Включает таймер в статусе при прослушивании "Волны"

wavetime = 0
Отключает таймер в статусе при прослушивании "Волны"

# Кнопка "Слушать"

button = 1
Включает кнопку

button = 0
Убирает кнопку
